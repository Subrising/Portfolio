Recent Projects and Accomplishments

Re-Do - Facial Recognition, Redaction and Replacement

Re-Do is a web application that allows a user to recognise individuals within images and based on the inputs provided, the recognised individuals can be redacted/replaced or every other face but theirs can be redacted/replaced.

This project was built using the PERN stack (PostgreSQL, Express, React, and Node.js) with the front-end consuming external APIs of Face-API.ks and JIMP, along with using my own back-end server API. The front-end, back-end and database are deployed on Heroku allowing for continuous integration and development.

Next 5 Races - LadBrokes

N5 is a web application that displays the next five races that can be bet on, sorted by the closing date. When a race is closed, you can no longer bet or view that race.

This was a dockerised project using Golang for the API & back-end development along with PostgreSQL for the database and Gin Gonic as the web framework.

SmartEye - RedEye Apps

SmartEye will allow RedEye’s clients to have greater control over the information contained in their files, giving them the ability to search through uploaded files for specific information and metadata along with allowing them to upload a greater variety of file types.

My part of the project is the data extractor of SmartEye, allowing users to upload files to the RedEye server which will then be converted to images and sent to Google Cloud Vision, with the results returned and stored in RedEye’s database. Clients/users will then be able to query RedEye’s servers for information on their files stored in the database.
